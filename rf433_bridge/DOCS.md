# Home Assistant App: RF433 Bridge

## Installation

Follow these steps to get the app (formerly known as add-on) installed on your system:

1. In Home Assistant, go to **Settings** > **Apps** > **Install app**.
2. Find the "RF433 Bridge" app and click it.
3. Click on the "INSTALL" button.

## How to use

1. Connect the RF433 transciever to a USB port.
2. Set the prot and baud rate in the configuration page.
3. Start the app.
4. Open the WEB UI.
5. As RF Devices are found, they will appear in the Live RF section. These can be added to the configuration.
